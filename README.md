# sm234

#### 介绍
近日有项目需要使用招行的聚合支付，需要使用国密sm2算法来签名和验签。因招行暂未提供php的代码，所以特搜集了需要用到的所有类库，并写了一个简单的文件来获取主动支付二维码，供有同样需求的小伙伴参考。
